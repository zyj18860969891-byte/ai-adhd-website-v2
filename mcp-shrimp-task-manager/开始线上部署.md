## 🔧 MCP客户端配置

### Cursor IDE配置
部署成功后，在Cursor设置中添加：

```json
{
  "mcpServers": {
    "shrimp-task-manager": {
      "command": "node",
      "args": [
        "/path/to/mcp-shrimp-task-manager/dist/custom-mcp-server.js"
      ]
    }
  }
}
```

### VS Code配置
在 `.vscode/settings.json` 中添加：

```json
{
  "mcp.serverProviders": {
    "shrimp-task-manager": {
      "command": "node",
      "args": [
        "/path/to/mcp-shrimp-task-manager/dist/custom-mcp-server.js"
      ]
    }
  }
}
```

## 📞 遇到问题？

### 常见问题快速解决

**问题1: 构建失败**
- 检查Railway日志确认错误信息
- 确保Node.js版本 >= 18.0.0
- 确认所有依赖正确安装

- 查看运行日志
- 确认端口3009未被占用
- 检查环境变量配置

**问题3: MCP客户端连接失败**
- 确认服务正在运行
- 检查文件路径是否正确
