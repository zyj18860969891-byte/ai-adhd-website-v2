## 🚨 故障排除

### 常见问题快速解决

1. **服务无法启动**
   ```bash
   # 检查Node.js版本
   node --version  # 应该 >= 18.0.0
   
   # 重新安装依赖
   rm -rf node_modules package-lock.json
   npm install
   
   # 重新构建
   npm run build:mcp
   ```

2. **MCP客户端连接失败**
   - 确认服务正在运行
